# MarbleMaze

**摘要**：
长安大学信息工程学院2021级本科生在课程学习过程中开发了一款手机游戏，游戏名称叫做“弹珠迷宫”。
![CoverImage](/intro.png)

**游戏内容**:
在一个随机生成的平面地图上，玩家不断蓄力弹动弹珠，使其在地图上滑动，通过调整合适的方向以及力度，绕过障碍物，在规定时间内将弹珠弹到目标位置，同时要避免将弹珠弹出地图范围。

**开发工具**:
Unity3D & C#
